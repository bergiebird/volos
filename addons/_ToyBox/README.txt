Welcome to the ToyBox
A Component Framework and Game Template
For Making 2D Games in Godot used for ease of collaboration and speed of prototyping.

This Template is not to be distrubuted or sold
As it uses Assets that are limited to commercial works made or purchased by BergieBird, Vrood, and contributors.
If you have come across this package without proper licencing, the framework may
be used but ALL ASSETS MUST BE SCRUBBED FROM THIS PROJECT IMMEDIATELY AND CANNOT BE USED FOR PERSONAL OR COMMERCIAL PURPOSES.

Use of this Framework is limited to the various licensing restrictions presented in the tools and scripts within.


Best Practice: After Completing a Project, scrub all traces of the Template away until only the Game remains and then ship
that. Using the whole Framework for shipping will add needless bloat and potential liability.
