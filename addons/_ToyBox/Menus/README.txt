Welcome to the Menu directory!

Here be where all of the "Default" Menu templates from Maacks exist, but ready to be customized for
your project, no need to go through the setup.

Read the docs in maacks game template addon for more information
TLDR, you'll need to make some themes and customize the AudioBus and fill in a few form fields


KNOWN ISSUES: The Menu Template isn't the biggest fan of having resolutions reset during Runtime. Probably worth setting it to
only resize on Gamestartup, requiring a reset for the resolution changes to take affect.
